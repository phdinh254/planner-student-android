# PersonalPlanner - Android Java XML SQLite

Ứng dụng quản lý kế hoạch cá nhân cho đồ án môn Lập trình thiết bị di động.

## Công nghệ
- Java
- XML Layout
- SQLite thuần
- SharedPreferences
- RecyclerView
- BottomNavigationView
- ExecutorService + Handler

## Chức năng
- Đăng ký tài khoản
- Đăng nhập
- Lưu phiên đăng nhập
- Đăng xuất
- Thêm kế hoạch
- Hiển thị danh sách kế hoạch
- Tìm kiếm kế hoạch
- Sửa kế hoạch
- Xóa kế hoạch có xác nhận
- Đánh dấu hoàn thành kế hoạch
- Xem lịch

## Cách mở project
1. Mở Android Studio.
2. Chọn **Open**.
3. Chọn thư mục `PersonalPlanner`.
4. Chờ Gradle Sync.
5. Chạy ứng dụng bằng máy ảo hoặc điện thoại thật.
